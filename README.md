# AlgoCore

A complete Data Structures and Algorithms (DSA) implementation in C++ with modular design and interactive CLI.

---

## 🚀 Features

### 🔹 Data Structures

* Singly Linked List

  * Insert (Head, Tail)
  * Reverse
  * Search
  * Find Middle
  * Cycle Detection

* Doubly Linked List

  * Insert (Head, Tail)
  * Forward & Backward Traversal
  * Delete Node (O(1))

---

### 🔹 Algorithms / Problems

* LRU Cache

  * Implemented using Doubly Linked List + HashMap
  * Supports `get()` and `put()` in **O(1)** time

---

## 🧠 Concepts Covered

* Pointers & Dynamic Memory
* Linked List (SLL & DLL)
* Two Pointer Technique
* HashMap (`unordered_map`)
* Cache Design (LRU Policy)
* Time Complexity Optimization
